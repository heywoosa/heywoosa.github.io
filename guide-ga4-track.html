GA4 追蹤教學：機票按鈕的點擊率分析

— 掌握這個技巧，才能知道你的網站是否真的賺錢！

一、為什麼要用 GTM 追蹤按鈕？

GA4 直接安裝雖然可以追蹤頁面瀏覽量，但無法知道使用者是否點擊了關鍵的 **「查機票」** 或 **「Klook 行程」** 按鈕。透過 Google Tag Manager (GTM)，我們可以精準發送「事件 (Event)」，這對優化聯盟行銷非常重要。

二、GTM 步驟：追蹤「點擊事件」

請先確保您已經將 GTM 代碼 (`GTM-MRN7GLNZ`) 貼入網站。

Step 1: 啟用點擊變數 (Variables)

在 GTM 後台：點擊 **「變數 (Variables)」** -> 在「內建變數」區塊點擊 **「設定 (Configure)」** -> 勾選 **「Click ID」** 和 **「Click Element」**。

Step 2: 建立觸發條件 (Triggers)

點擊 **「觸發條件 (Triggers)」** -> **「新增 (New)」** -> 選擇類型 **「僅限連結 (Just Links)」**。

  • 將觸發條件設為：**「部分連結點擊 (Some Link Clicks)」**。
  • 條件設定：`Click ID` **包含** `calcBtn` 或 `flightLink`。

Step 3: 建立 GA4 代碼 (Tags)

點擊 **「代碼 (Tags)」** -> **「新增 (New)」** -> 選擇類型 **「Google Analytics (GA4) 事件」**。

  • **事件名稱**：設為 `affiliate_click`。
  • **觸發條件**：選擇剛剛建立的觸發條件。

⭐ 提交 (Submit) 後，你就能在 GA4 報表裡看到每一筆機票按鈕的點擊次數了！
